# Kommerzielle Lizenz für EhrenZeit

## Übersicht
Diese kommerzielle Lizenz erlaubt die Nutzung von EhrenZeit außerhalb der 
Bedingungen der AGPL-3.0-Lizenz.

## Wer benötigt eine kommerzielle Lizenz?

- **SaaS-Anbieter**: Hosting als kostenpflichtige Dienstleistung
- **Systemhäuser**: Integration in kommerzielle Produkte
- **Beratungsunternehmen**: Weiterverkauf mit proprietären Erweiterungen
- **Unternehmen**: Nutzung ohne Offenlegung eigener Änderungen

## Nutzungsrechte

Die kommerzielle Lizenz gewährt:
- ✓ Nutzung in geschlossenen/proprietären Systemen
- ✓ Keine Pflicht zur Veröffentlichung von Änderungen
- ✓ Hosting als kostenpflichtige SaaS-Lösung
- ✓ Integration in kommerzielle Produkte
- ✓ White-Label-Lösungen
